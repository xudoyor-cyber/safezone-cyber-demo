What is SQL Injection?

SQL Injection is a type of attack where hackers input malicious SQL code into forms to access or destroy your database.

Imagine you enter this into a login form:

' OR '1'='1

This tricks the database into logging you in without knowing the password!

Why it's dangerous

  • Attackers can steal all user data
  • They can delete or change your database
  • Your website can be fully taken over

How to protect your site

  • Always use prepared statements (as shown in secure_login.py)
  • Never trust user input
  • Validate and sanitize all data
